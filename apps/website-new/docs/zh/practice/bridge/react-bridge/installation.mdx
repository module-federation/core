# 安装

要开始使用 React Bridge，您需要安装 `@module-federation/bridge-react` 包。

import { PackageManagerTabs } from '@theme';

<PackageManagerTabs
  command={{
    npm: 'npm install @module-federation/bridge-react@latest',
    yarn: 'yarn add @module-federation/bridge-react@latest', 
    pnpm: 'pnpm add @module-federation/bridge-react@latest',
  }}
/>

## 版本要求

- **React**: 16.x - 19.x
- **Node.js**: >= 16.0.0
- **Module Federation**: >= 2.0.0

## TypeScript 支持

React Bridge 完全支持 TypeScript，内置了完整的类型定义。无需额外安装类型包。

```tsx
import { createBridgeComponent, createRemoteAppComponent } from '@module-federation/bridge-react';

// TypeScript 会自动推断类型
const bridgeComponent = createBridgeComponent({
  rootComponent: App
});
```

安装完成后，您可以继续阅读 [导出应用](./export-app) 来了解如何导出您的 React 应用。
