## 原因

当生产者入口文件被正常加载后，可以在全局对象（globalThis）通过 remoteEntryKey 获取对应的模块导出。
然而在本次加载过程中，无法获取。该错误存在三种可能原因：

1. remoteEntryUrl 不正确
2. remoteEntryKey 不正确
3. 生产者入口文件没有正常挂载容器

## 解决方法

针对原因，有对应的解决方法：

1. 检查生产者入口地址是否正确
  如果使用的是 manifest ，那么检查 manifest 中的 publicPath 和 remoteEntry.name 字段
2. 检查 remoteEntryKey 字段是否正确
  - 若remoteEntry是 js 文件，在 js 文件中搜索 remoteEntryKey 的值，看是否存在
  - 若使用的是 manifest ，检查 manifest 中的 globalName 字段是否和 remoteEntryKey 值一致
3. 若项目构建器为 rspack ，查看最终构建配置是否设置了 [runtimeChunk](https://rspack.dev/config/optimization#optimizationruntimechunk) ，若是则删除此配置
