# Overview

{ props.name || 'Module Federation' } 为不同构建器提供了配套的构建插件。选择与你项目构建器对应的插件文档，即可完成接入与配置。

不同构建器的插件可能包含少量专属选项，但核心的配置项保持一致，详见[配置项](../../../configure/index)。
