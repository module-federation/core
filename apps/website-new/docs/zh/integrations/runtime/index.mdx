# Runtime

如果你的项目不方便接入构建插件，或者只需要在运行时动态注册和加载远程模块，可以从运行时方式开始。

运行时 API、模块注册、模块加载、运行时插件等能力，请继续阅读 [Runtime 接入](/guide/runtime/index)。

这种方式适合特殊接入场景。大多数 Web 应用仍然建议优先选择当前项目对应的接入方案，例如 [Rsbuild](/integrations/build-tool/rsbuild)、[Rspack](/integrations/bundler/rspack)、[Webpack](/integrations/bundler/webpack) 或 [Modern.js](/integrations/framework/modernjs)。
