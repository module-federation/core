# treeShakingSharedExcludePlugins

Configure plugin names to exclude during the shared dependency tree-shaking/fallback build process.

- Type: `string[]`
- Required: No
- Default: `['HtmlWebpackPlugin', 'HtmlRspackPlugin']`

This option lets you specify a list of plugin names that should be ignored (not applied) when building shared dependency tree-shaking/fallback artifacts.
